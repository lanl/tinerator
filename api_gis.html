GIS Module Functions¶

Loading files¶

tinerator.gis.load_raster(filename: str, no_data: Optional[float] = None)

Loads a raster from a given filename. Supports all raster datatypes that GDAL supports, including GeoTIFF, IMG, and ASC.

Parameters

  • filename (str) – The input filename.

  • no_data (float, optional) – If NoData value is not encoded in the raster, it can be manually set here.

  • to_crs (str, optional) – If provided, will reproject raster to the given CRS (can be an EPSG code, WKT string, or pyproj.CRS object.)

Returns

A tinerator.Raster object.

Examples

>>> dem = tin.gis.load_raster("my_dem.tif", no_data=-9999., crs="EPSG:3114")
tinerator.gis.load_shapefile(filename: str) → tinerator.gis.geometry.Geometry

Given a path to a shapefile, reads and returns each object in the shapefile as a tinerator.gis.Shape object.

Parameters

filename (str) – The input filename.

Returns

A tinerator.gis.Geometry object.

Examples

>>> boundary = tin.gis.load_shapefile("my_shapefile.shp", crs="EPSG:3114")

Reprojection¶

tinerator.gis.reproject_raster(raster: tinerator.gis.raster.Raster, crs: Union[pyproj.crs.crs.CRS, str, dict, int]) → tinerator.gis.raster.Raster¶

Reprojects a TINerator Raster object into the destination CRS.

The CRS must be a pyproj.CRS object, a WKT string, an EPSG code (in the style of “EPSG:1234”), or a PyProj string.

Parameters

  • raster (Raster) – A TINerator Raster object to reproject.

  • dst_crs (pyproj.CRS) – The CRS to project into.

Returns

A reprojected Raster object.

Examples

>>> tin.gis.reproject_raster(dem, "EPSG:3857")
tinerator.gis.reproject_geometry(shape: tinerator.gis.geometry.Geometry, crs: Union[pyproj.crs.crs.CRS, str, int, dict]) → tinerator.gis.geometry.Geometry¶

Reprojects a Geometry object into the provided CRS and returns the new object.

Parameters

  • shape (Geometry) – The Geometry object to reproject.

  • crs (Union[pyproj.CRS, str, int, dict]) – The CRS to reproject into.

Returns

A TINerator Geometry object in the new coordinate reference space.

Resample a Raster¶

tinerator.gis.resample_raster(raster: tinerator.gis.raster.Raster, new_res: Optional[tuple] = None, new_shape: Optional[tuple] = None, resampling_method: str = 'near') → tinerator.gis.raster.Raster¶
Resamples a raster into either/both of:

  • A new resolution in geospatial units, as (xRes, yRes)

  • A new raster shape, in (rows, cols)

Various resampling algorithms are available on the GDAL website.

https://gdal.org/programs/gdalwarp.html#cmdoption-gdalwarp-r

Parameters

  • raster (Raster) – A TINerator Raster object.

  • new_res (Tuple[float, float], optional) – The new resolution in geospatial coordinates.

  • new_shape (Tuple[int, int], optional) – The new number of rows and columns for the raster.

Returns

A TINerator Raster object

Examples

>>> new_shape = (x//2 for x in dem.shape)
>>> new_dem = tin.gis.resample_raster(dem, new_shape = new_shape)

Clipping a Raster with Geometry¶

tinerator.gis.clip_raster(raster: tinerator.gis.raster.Raster, shape: tinerator.gis.vector.Shape) → tinerator.gis.raster.Raster¶

Returns a new Raster object, clipped by a Shape polygon. The raster will have the same shape and projection as the previous one, but cells not covered by shape will be filled with the no data value of the parent raster.

Parameters

  • raster (tinerator.gis.Raster) – The raster to clip.

  • shape (tinerator.gis.Shape) – The shape to clip the raster with.

Returns

A clipped raster.

Examples

>>> dem = tin.gis.load_raster("raster.tif")
>>> boundary = tin.gis.load_shapefile("boundary.shp")
>>> new_dem = tin.gis.clip_raster(dem, boundary)

Generate flowlines from a DEM using watershed delineation¶

tinerator.gis.watershed_delineation(raster: tinerator.gis.raster.Raster, threshold: Optional[float] = None, method: str = 'D8', exponent: Optional[float] = None, weights: Optional[numpy.ndarray] = None, return_matrix: bool = False) → Tuple[tinerator.gis.geometry.Geometry, Optional[numpy.ndarray]]¶

Performs watershed delination on a DEM. Optionally, fills DEM pits and flats.

Parameters

  • raster (tin.gis.Raster object) – A DEM as a TINerator Raster object.

  • threshold (float, optional) – The numerical threshold where every cell with a value above this number is considered a flow path, and every cell under this number is not.

  • method (str, optional) – The watershed delineation algorithm, one of: * D8,

Returns

river_network – The extracted river network.

Return type

tin.gis.Shape

Examples

>>> d = tin.gis.load_raster("dem.tif")
>>> rn = tin.gis.watershed_delineation(d, 500.)
>>> d.plot(layers=[rn])

Converting a triangulation into a Geometry object¶

tinerator.gis.vectorize_triangulation(mesh, exodus_mapping: bool = True)¶

Saves triangulated surface as an ESRI Shapefile. Contains additional fields of:

  • “elementID” - int - the integer ID of the triangle

  • “materialID” - int - the material ID of each triangle

  • “elevation” - float - triangle centroid Z value

Useful for importing into QGIS / ArcGIS to validate the triangulation relative to other GIS objects.

Parameters

  • (str) (outfile) –

  • (tinerator.meshing.Mesh) (mesh) –

Note

Only triangulations will work with this function. Prism meshes and other mesh types will not.

Examples

>>> surface_mesh = tin.meshing.triangulate(dem, min_edge_length=0.1)
>>> triang_geom = tin.gis.vectorize_triangulation(surface_mesh)
>>> triang_geom.save("triangulation.shp")

Rasterize Geometry¶

tinerator.gis.rasterize_geometry(raster: tinerator.gis.raster.Raster, shape: tinerator.gis.vector.Shape) → tinerator.gis.raster.Raster¶

Rasterizes a Geometry object into a raster similar to raster.

Parameters

  • raster (Raster) – The TINerator Raster object to use as a template for rasterizing the shape.

  • shape (Geometry) – The TINerator Geometry object to rasterize.

Returns

A TINerator Raster object with the Geometry burned into a raster of the same extent, projection, and size as raster.

Examples

>>> rasterized_shape = tin.gis.rasterize_geometry(dem, watershed_flowline)

Create a pyproj.CRS object¶

tinerator.gis.parse_crs(crs: Union[str, int, dict]) → pyproj.crs.crs.CRS¶
Returns a pyproj.CRS object from:

  • A string (Proj4 string, “epsg:xxxx”, Wkt string, …)

  • An int (EPSG code)

  • A dict with Proj4 projection

See the Proj4 documentation for more information.

https://pyproj4.github.io/pyproj/dev/api/crs/crs.html

Parameters

crs (Union[str, int, dict]) – A PyProj4-supported projection.

Returns

A pyproj.CRS object.

Examples

>>> tin.gis.parse_crs(26915)
>>> tin.gis.parse_crs("EPSG:26915")
>>> tin.gis.parse_crs("+proj=geocent +datum=WGS84 +towgs84=0,0,0")

Compute the distance field from a Geometry object to all Raster cells¶

tinerator.gis.distance_map(raster: tinerator.gis.raster.Raster, shape: tinerator.gis.geometry.Geometry, min_dist: float = 0.0, max_dist: float = 1.0) → tinerator.gis.raster.Raster¶

Creates a distance map. A new raster will be returned, where every cell will contain the (normalized) distance to the nearest intersection on shape.

By default, the values will be normalized from 0 to 1. Setting min_dist and max_dist adjust that range.

Parameters

  • raster (tinerator.gis.Raster) – The input raster.

  • shape (tinerator.gis.Shape) – The shape to measure distance from.

  • min_dist (float, optional) – Defaults to 0.

  • max_dist (float, optional) – Defaults to 1.

Returns

A raster filled with the normalized distance from each cell to the nearest intersection on shape.

Note

It is rare that a user would want to call this. This is mainly an internal function used for triangulation.

Examples

>>> dist = tin.gis.distance_map(dem, flowline)
