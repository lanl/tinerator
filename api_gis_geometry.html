Geometry object¶

Loading a Shapefile into a Geometry object¶

tinerator.gis.load_shapefile(filename: str) → tinerator.gis.geometry.Geometry¶

Given a path to a shapefile, reads and returns each object in the shapefile as a tinerator.gis.Shape object.

Parameters

filename (str) – The input filename.

Returns

A tinerator.gis.Geometry object.

Examples

>>> boundary = tin.gis.load_shapefile("my_shapefile.shp", crs="EPSG:3114")

Geometry class member functions¶

class tinerator.gis.Geometry(shapes: Optional[list] = None, crs: Optional[pyproj.crs.crs.CRS] = None, properties: Optional[collections.OrderedDict] = None)¶

Creates a Geometry object. This object stores a collection of shapes in Shapely format (under Geometry.shapes), along with a CRS and individual shape attributes.

Refer to the Shapely documentation at https://shapely.readthedocs.io for the various methods that Shapely objects support.

add_property(name: str, data: list, type: Optional[str] = None)¶

Adds a property to the Geometry object.

Parameters

  • name (str) – The name of the property.

  • data (list) – A list containing the property data. Must be of length len(self.shapes).

  • type (str, optional) – The type of the data. Must be str, int, float, or date.

property centroid¶

Returns the centroid of the Geometry object.

property coordinates¶

Returns the (x, y) array of all shapes in the Geometry object.

property extent¶

Returns the spatial extent of the Geometry object in the form (xmin, ymin, xmax, ymax).

property geometry_type¶

Returns the geometry type of this object in GeoJSON format.

property ndim¶

The number of dimensions of the Geometry object. May be 2 or 3.
