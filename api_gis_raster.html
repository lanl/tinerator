Raster object¶

Loading a raster from a file¶

tinerator.gis.load_raster(filename: str, no_data: Optional[float] = None)¶

Loads a raster from a given filename. Supports all raster datatypes that GDAL supports, including GeoTIFF, IMG, and ASC.

Parameters

  • filename (str) – The input filename.

  • no_data (float, optional) – If NoData value is not encoded in the raster, it can be manually set here.

  • to_crs (str, optional) – If provided, will reproject raster to the given CRS (can be an EPSG code, WKT string, or pyproj.CRS object.)

Returns

A tinerator.Raster object.

Examples

>>> dem = tin.gis.load_raster("my_dem.tif", no_data=-9999., crs="EPSG:3114")

Raster class member functions¶

class tinerator.gis.Raster(raster_path: str, no_data: Optional[float] = None)¶

The main Raster class object in TINerator. This stores the data and metadata for a loaded or created Raster object (like from reading in a GeoTIFF), and has many helper functions for reprojection, resampling, depression filling, and more.

property area¶

Returns the surface area of the raster, ignoring noDataValue cells.

property centroid¶

Returns the raster centroid.

property extent¶

Returns the spatial extent of the raster, in the form:

(x_min, y_min, x_max, y_max)

fill_depressions(fill_depressions: bool = True, fill_flats: bool = True)¶

Fills flats and depressions in a DEM raster. On meshes intended to be high-resolution, leaving flats and depressions untouched may cause solver issues. This method should be called before generating a triangle mesh from a DEM raster.

# Arguments fill_depressions (bool): fill pits and depressions on DEM fill_flats (bool): fill flats on DEM

property geotransform¶

Raster geotransform. In the form of: (x_min, pixel_width, 0, y_min, 0, pixel_width)

get_boundary(distance: Optional[float] = None) → tinerator.gis.geometry.Geometry¶

Get a line mesh with nodes seperated by distance distance that describe the boundary of this raster object.

property mask¶

Returns a boolean mask where the raster contains no_data_value elements.

masked_data()¶

Returns a copy of the raster data where all no_data_value elements in the raster are replaced with numpy.nan.

property max_cell¶

Returns the first cell index representing the largest value in the raster.

property min_cell¶

Returns the first cell index representing the smallest value in the raster.

property origin¶

Returns the lower left corner of the raster.

plot(layers: Optional[list] = None, outfile: Optional[str] = None, title: Optional[str] = None, geometry: Optional[list] = None, hillshade: bool = False)¶

Plots the raster object.

# Arguments outfile (str): path to save figure title (str): figure title geometry (list): a list of geometrical objects to overlay

save(outfile: str)¶

Saves a raster object to disk in GeoTIFF format. Writes array as Float64.

value_at(x: float, y: float)¶

Returns the value of the raster at point (x, y) in the same CRS as the raster.

values_at(points: numpy.ndarray)¶

Returns the raster values at points, where points is a coordinate X-Y array in the same CRS as the raster.
