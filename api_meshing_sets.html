Surface mesh generation¶

Generating a surface mesh¶

tinerator.meshing.Mesh.surface_mesh(self)¶

Extracts the surface mesh: a mesh where all interior voxels and faces are removed. For triangular meshes, this will return a line mesh. For a prism (extruded) mesh, this will return a mesh containing triangles and quads.

Returns in VTK/PyVista format.

Returns

surface_mesh

Return type

The surface (exterior) mesh.

discretize_sides(geom: numpy.array, close_ends: bool = True, at_layer: Optional[tuple] = None, set_name_prefix: str = 'Sides')¶

Discretizes the side external faces of the mesh with the given coordinates.

Coordinates must be given in clockwise ordering.

:param geom (np.ndarray): :type geom (np.ndarray): a list of (x, y) coordinates :param close_ends (bool: :type close_ends (bool: If True, discretizes the entire perimeter :param optional): :type optional): If True, discretizes the entire perimeter :param at_layer (tuple: :type at_layer (tuple: Generate a set from a specified layer :param optional): :type optional): Generate a set from a specified layer :param set_name_prefix (str: :type set_name_prefix (str: The set name prefix :param optional): :type optional): The set name prefix

Returns

sets

Return type

a list of side sets.

Examples

>>> sides = [
    (65.5, 0.6),
    (50.5, 0.04),
    (12.0, 0.04),
    (12.0, 12.9),
    (79.1, 9.1),
]
>>> outlet = [
    (71.9, 4.7),
    (67.3, 2.1),
]
>>> sides_side_set = sf.discretize_sides(sides)
>>> outlet = sf.discretize_sides(
        outlet,
        close_ends = False,
        at_layer = (1, 1),
        set_name_prefix="Outlet"
    )

from_geometry(geometry, on_top: bool = True, set_name: str = 'Geometry')¶

From a TINerator Geometry object, returns a set of all top-surface faces that intersect with the Geometry object.

As an example, this can be used to capture flowline cells generated via watershed delineation.
